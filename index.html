Connect Cloudflare (one-click Protect)

Paste a Cloudflare API Token (Permissions: Zone:Read and DNS:Edit, scoped to the zone you’ll test). Nothing is stored—token is used only for this request.

Paste token → List zones → choose zone → Protect.

Connect AWS Route 53 (one-click Protect)

Paste an AWS Access Key ID + Secret (IAM user limited to Route 53). Nothing is stored—keys are used only for this request.

Paste keys → List hosted zones → choose one → Protect.
